# This test is used to test whether the API of each mode of dmadim is normal.

### introduce
+ Test memset function
+ Test checksum function
+ Test 1D/D/3D/4D copy function
+ Test the synchronous and asynchronous modes of each function

### use
+ sample path and name map/out/bin/sample_dmadim
+ Execution method ./sample_dmadim
+ If the test passes, the terminal will print "test successful.". If the test fails, "test failed!" and fail function information, line number, etc. will be printed.
